DIRECTORY: ./library
_________________________________________________________

/hysplit

	Subroutines required to create the HYSPLIT libraries
 	libhysplit.a and libhysplitcxx.a are needed by most
        programs in the ./source directory.

/fcsubs

	Subroutines required for reading variable length
	records as found in many GRIB data files.

/lbfgsb

	Subroutines required for the cost function minimization 
	program.
